# Scott Logic Games

###Setup

`npm install`

###Running

`npm start`
        
###Development

`npm run dev`

This project uses gulp 4 so to run individual gulp tasks you'll need to uninstall previous versions of gulp with `npm uninstall gulp -g` and install gulp 4 with `npm install gulpjs/gulp-cli#4.0 -g`

###Release

`npm run release`
